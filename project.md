# 背景
我们团队在亚马逊上销售产品。

# 目标
我们团队希望通过一个 Dashboard 来展示我们产品在亚马逊上的销售情况。

# 要求
1. 团队中的每个成员都能看到，尤其是老板和运营，因此需要部署到互联网上。
2. 主要关注当前销售指标、目标，以及两者之间的差距、完成进度。
3. 请你自行搜索 Dashboard 方面（销售 / 亚马逊销售 / BI）目前 Skill / Plugin 的最佳实践，并加以使用。
4. 目前主要关注我们正在主推的一个产品（ASIN 为 B0GXYYZPBW），你可以通过 lingxing mcp 获取到这个产品销售和广告方面的数据
5. 同时也要关注这个产品除了销售和广告以外的其他上下文，这部分你可以参考下载文件夹下的 amazon-ad-agent-workspace-3-redesign 的 skills 文件的 decision-context-capturer。
